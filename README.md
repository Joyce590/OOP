# OOP Python Basics

This project contains simple examples of Object-Oriented Programming  in Python and different examples used in real experiences

## Topics Covered
- Classes in oop and examples
- Objects in oop and examples
- Constructors in oop and examples
- Destructors in oop and examples

## Files
- classes.py which  shows a simple class
- objects.py which shows how objects work
- constructors.py  which explains constructors
- destructors.py which explains destructors
